Enter Your Google Maps API Key

This game requires a Google Maps API key to play.

How do I get a key?
  1. Go to the Google Cloud Console and create a project.
  2. In the navigation menu, go to APIs & Services > Library.
  3. Search for and enable the "Maps JavaScript API".
  4. Go to APIs & Services > Credentials, click "Create Credentials," and select "API Key."
  5. Copy the key and paste it above.
